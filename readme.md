\f0\fs24 \cf0 # Buctril Field Engagement Report\
\
Interactive dashboard for Bayer Buctril wheat-field engagement sessions.\
\
## Live report\
\
\uc0\u55357 \u56393  https://paksaf.github.io/Buctril-Field-Report/\
\
- Auto-loads data from **`sum-sheet.csv`** (export from Excel).\
- Renders an interactive Leaflet map with session markers.\
- Shows KPIs, day-wise breakdown, and charts via Chart.js.\
- Wheat-field video loop (**`tmp.mp4`**) runs softly blurred in the background.\
\
To update the report, simply upload a new `sum-sheet.csv` with the same column headers and GitHub Pages will serve the updated dashboard.\
